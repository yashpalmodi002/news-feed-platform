## Design Notes

### Color Scheme
- Primary: Blue (#2563EB)
- Secondary: Gray (#64748B)
- Success: Green (#10B981)
- Background: White (#FFFFFF)
- Text: Dark Gray (#1E293B)

### Typography
- Headings: 'Inter', sans-serif
- Body: 'Inter', sans-serif
- Font sizes: 
  - H1: 32px
  - H2: 24px
  - Body: 16px
  - Small: 14px

### Components
1. **Navigation Bar**: Fixed top, includes logo, search, notifications, user menu
2. **Topic Pills**: Horizontal scrollable tabs for category filtering
3. **Article Cards**: Image, title, source, summary, action buttons
4. **Buttons**: Rounded, filled primary or outlined secondary
5. **Icons**: Using Font Awesome or Heroicons

### Interactions
1. **Hover**: Cards have subtle shadow lift on hover
2. **Click**: Article cards expand or navigate to detail page
3. **Save**: Toggle button to save/unsave articles
4. **Share**: Opens share modal with social media options
5. **Infinite Scroll**: Auto-load more articles as user scrolls

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
